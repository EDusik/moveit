## :computer: Commands
### Create the project
#### React
```nmp
npm install typescript @types/react @types/react-dom @types/node -D
```
#### Next.Js
```npm
npx create-next-app moveit-next
```
### Install Dependencies 
#### React
```nmp
npm install
```
#### Next.Js
```npm
npm install typescript @types/react @types/react-dom @types/node -D
```
### Run Project
#### React
```nmp
npm start
```
#### Next.Js
```npm
npm run dev
```
