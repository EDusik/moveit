## :computer: Commands
### Create the project

```npm
npx create-next-app moveit-next
npx create-react-app moveit --template=typecript
```
### Install Dependencies 
```npm
npm install typescript @types/react @types/react-dom @types/node -D
npm install
```
### Run Project
```npm
npm dev
npm start
```
